Git

1.初始化

git init

2.提交文件

git add 文件 [后面可以跟很多个文件]

*git commit -m “本次修改的注释” *

一次 git commit 可以一次性提交所有的git add

git分为暂存区、工作区、版本区

一个 git add命令实际上就是把要提交的所有修改放到暂存区（Stage），然后，执行git commit就可以一次性把暂存区的所有修改提交到分支。
